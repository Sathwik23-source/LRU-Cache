LRU Cache

What is LRU Cache?

An LRU (Least Recently Used) Cache is a type of cache algorithm that discards the least recently used items first when the cache reaches its capacity. It’s commonly used to manage memory efficiently.

⚙️ How It Works

  • 🔹 It keeps track of the usage order of the items.
  • 🔹 When you access (get) or insert (put) an item, it's marked as the most recently used.
  • 🔹 If the cache exceeds its capacity, it evicts the least recently used item.
